# Security Policy

## Supported Versions

This project is pre-1.0 and receives security updates on the latest release line only.

## Reporting a Vulnerability

Please report suspected vulnerabilities privately via GitHub Security Advisories:

- https://github.com/apvarun/guard-sdk/security/advisories/new

Do not open a public issue for unpatched vulnerabilities.

Include the following when possible:

- Affected package(s) and version(s)
- Reproduction steps or proof of concept
- Impact assessment
- Any known mitigations

We will acknowledge reports as quickly as possible and coordinate disclosure after a fix is available.
